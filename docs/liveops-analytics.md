# Live Ops Analytics

Operational events (no emails, tokens, receipts, or private payloads):

- `remote_config_fetched` / `remote_config_failed`
- `maintenance_screen_viewed` / `update_required_viewed`
- `season_viewed` / `season_reward_claimed`
- `event_viewed` / `event_joined` / `event_completed` / `event_reward_claimed`
- `announcement_viewed` / `announcement_action_tapped`
- `report_submitted`
- `run_validation_rejected`
- `purchase_fulfillment_delayed`

Dashboards should label incomplete metrics as unavailable. Do not invent LIVE numbers.

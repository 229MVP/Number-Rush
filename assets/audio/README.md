# Number Rush audio assets

All files under `sfx/` and `music/` are **original placeholder synthesized WAV tones** generated by:

```bash
node scripts/generateAudioAssets.mjs
```

They are not copied from any commercial game. Replace with production art/audio before shipping.

|------|-----|

| screenOpen.wav | Screen transitions |

| File | Use |
